# clogii-react-native

## Project includes
1. react-native-fbsdk
1. react-native-fcm
  
## Installation
```
npm i
cd ios && pod install
```

## Build with Platform
- iOS : Open `.xcworkspace` and test with xcode
- Android : `$ react-native run-android`
